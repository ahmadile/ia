# 💰 Personal Budget Analyzer

## 📌 Description
A simple Python script that allows users to track their expenses and calculate their remaining budget.

## 🛠️ Skills Demonstrated
- Variables & Data Types
- Lists & Dictionaries
- While Loops (user input collection)
- For Loops (data processing)
- Conditional Statements (if/else)

## 🚀 How to Run
1. Clone the repository
2. Run: `python budget_analyzer.py`
3. Enter your expenses one by one
4. Type 'quit' to see your budget summary

## 📚 Learning Context
This project was built using only Python fundamentals (Chapters 1-3 of DataCamp's Introduction to Python for Developers).
No functions, no classes, no external libraries.
